💛
自分のことは、全て自分自身で管理する
この覚悟が凄く大切です。
あなたの人生は、全てあなたのものです。
他の人のせいに出来ることなどひとつもありませんよ。

С транскрипцией:
自分（じぶん）のことは、全（すべ）て自分自身（じぶんじしん）で管理（かんり）する
この覚悟（かくご）が凄（すご）く大切（たいせつ）です。
あなたの人生（じんせい）は、全てあなたのものです。
他（ほか）の人（ひと）のせいに出来（でき）ることなどひとつもありませんよ。

Перевод:
Очень важно быть готовым взять всё в свои руки. Твоя жизнь полностью принадлежит только тебе. Нет ни единой вещи, за которую можно было бы переложить ответственность на других.
